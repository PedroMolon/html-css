Citações

Para criar uma citação, usamos a tag Q, essa técnica é usada em citações no meio de parágrafos

Para criar uma citação mais longa (em bloco), usa-se a tag BLOCKQUOTE, ainda podemos colocar um CITE com o link da citação

Abreviações

Texto Invertido

Para colocar um texto invertido, usa-se a tag BDO, RTL = Da direita para a esquerda, LTR = Da esquerda para direita, Exemplo: Esse texto está invertido
