Listas Ordenadas

Uma lista ordenada, segue uma ordem, para isso utiliza-se a tag OL para delimitar a lista e LI para idetificar cada item

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

A tag OL tem um parâmetro type, que pode ser 1=listas numeradas, A=listas alfabéticas, a=alfabéticas minúsculas, I=algarismos romanos, i=algarismos romanos em minusculo

Pode também indicar o início da contagem com o parâmetro START

  1. Item
  2. Item
  3. Item

Listas não Ordenadas

Uma lista não ordenada, são aquelas que a ordem não influenciára no significado da lista, utiliza-se a tag UL

  • Item 1
  • Item 2
  • Item 3

Podemos definir parâmteros também, como disc, circle, square...


Listas Mistas

Podemos também criar listas mistas

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5

Lista de Definições

Uma lista de defiições é basicamente uma lista onde teremos o termo e a definição, para isso utilziamos as tags DL, DT e DD

HTML
Linguagem de marcação utilizada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites
JavaScript
Linguagem de programção para criar interatividades em sites
