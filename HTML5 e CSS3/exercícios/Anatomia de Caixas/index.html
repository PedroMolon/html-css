Configuração de Caixa

Height = Altura

Width = Largura

Background-color = Cor de fundo

Border = Borda, que cerca o conteúdo

Padding = Espaço interno entre a borda e o conteúdo

Margin = Espaço externo (margem) a partir da borda

Outline = Contorno dentro da margem, ao redor da borda

Tipos de Caixa

Block-level

Exemplo de caixa block-level

Pula para a linha de baixo, ocupa 100% da largura da tela, exemplo é a tag DIV

Inline-level

Exemplo de caixa inline-level-Continua na mesma linha, e não ocupa 100% da largura, exemplo é a tag SPAN

Shorthands

É uma maneira de facilitar

Border = largura, tipo, cor

Outline = largura, tipo, cor

Padding = topo, direita, baixo, esquerda

Margin = topo, direita, baixo, esquerda

Para colocar configuração de mesmo tamanho

Use apenas um valor

Para colocar configuração de 2 medidas

Use dois valores, o primeiro (topo e base) e o segunda (esquerda e direita)

Display

É um comando para modificar o tipo de caixa, para block ou inline level
