Imagens Flexíveis

Para poder adaptar o contúdo ao tamanho da tela, precisamos gerar imagens de tamanho diferentes e a fazer o navegador carregar a imagem certa para cada situação, para isso utilizamos as tags PICTURE e SOURCE.

Type= Tipo de imagem utilizada, srcset= Vai configurar o nome da imagem que será carregada qunado o tamanho indicado for atingido, media= indica o tamanho máximo a ser considerado para carregar a imagem indicada no atributo SRCSET.


Áudio

Para utilziar audios é necessário utilizar a tag AUDIO

Atributos da tag Audio

PRELOAD= indica se o áudio será pré-carregado ou não e aceita três valores, metadata= vai carregar apenas as informações sobre o arquivo, none= não vai carregar nada até clicar no play, auto= vai carregar o arquivo de áudio inteiro, assim que a página for carregada.

CONTROLS= vai apresentar o player na tela

AUTOPLAY= quando inserido, vai iniciar a reprodução do áudio assim que a página for carregada

LOOP= vai fazer que o áudio seja repetido eternamente

Vídeos

Para poder colocar vídeos, utilizamos a tag VIDEO, caso o arquvio seja hospedado em nosso próprio servidor

Atributos da tag Video

WIDTH= vai indicar a largura que o vídeo vai ter na tela

POSTER= condfigura uma imgem que vai aparecer como uma capa

CONTROLS= vai configurar se os controles do vídeo vão aparecer

AUTOPLAY= diz para o navegador se o vídeo vai começar a troca automaticamente

Vídeo do Youtube

Para colocar vídeo hospedado no youtube, deve pegar o código pronto ja no youtube, dentro da tag IFRAME
